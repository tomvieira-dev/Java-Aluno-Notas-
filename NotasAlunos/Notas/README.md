## Introdução

Este projeto é uma **aplicação Java de console** criada para praticar leitura de dados, uso de arrays, estruturas de repetição e validação básica de entradas.

A aplicação solicita ao usuário a quantidade de notas, aceita valores decimais usando **vírgula ou ponto** como separador, valida o intervalo das notas (0 a 10), calcula a média e exibe os resultados.

## Estrutura de Pastas

O workspace contém as seguintes pastas:

* `src`: pasta onde ficam os arquivos fonte

Os arquivos compilados são gerados na pasta `bin` por padrão.

```
src/
 └── notas/
     └── NotasAlunos.java
```

## Funcionamento

* Lê a quantidade de notas usando `Scanner`
* Armazena as notas em um array `double[]`
* Aceita entradas com `,` ou `.` como separador decimal
* Valida notas no intervalo **0 a 10**
* Calcula a média das notas
* Exibe a média e todas as notas informadas

## Executando a Aplicação

1. Abra o projeto no **VS Code**
2. Execute a classe `NotasAlunos`
3. Siga as instruções exibidas no terminal
